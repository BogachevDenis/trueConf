# trueConf
## Запуск проекта:
Для запуска приложения выполните следующие команды:
<br>
$ git clone git@github.com:BogachevDenis/trueConf.git
<br>
$ cd trueConf
<br>
$ go run main.go
<br>
#### Приложение запустится на http://localhost:8081/
## Запуск тестов:
$ cd trueConf
<br>
$ go test
